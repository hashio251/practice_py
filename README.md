# practice_py

Python の学習用リポジトリです。  
参考書を読みながら改変・作成したコードや、大学の授業で練習したコードをまとめています。

## ディレクトリ構成

```
practice_py/
├── python1/   # 参考書『Python1年生 第2版』ベースの学習コード
├── zen/       # 大学の授業で作成した演習コード（セクション別）
└── README.md
```

### python1/

『**Python1年生 第2版 体験してわかる！会話でまなべる！プログラミングのしくみ**』（翔泳社）を参考に、学習しながら改変・作成したコードを置いています。  
詳細は [`python1/README.md`](python1/README.md) を参照してください。

### zen/

大学の授業で取り組んだ Python の演習コードをセクションごとに整理しています。  
ファイル操作、Web スクレイピング、データ可視化など幅広いトピックを扱っています。

## 実行方法

Python 3 が必要です。

```bash
# 例: python1 ディレクトリのファイルを実行
python python1/bmi.py

# 例: zen ディレクトリのファイルを実行
python zen/section11/lesson1-write-txt.py
```
